## How to update content
All website text is in `index.html`. Search for the section you want to edit:
- **Hero / name / tagline** — search for `hero-name`
- **About paragraphs** — search for `about-text`
- **Stats (5+, 0%, $2M+)** — search for `stat-number`
- **Projects** — search for `proj-row`
- **Experience timeline** — search for `tl-item`
- **Contact email** — search for `macdevelopmentco.com`

## How to deploy
1. Push changes to the `main` branch
2. GitHub Pages automatically rebuilds — live in ~60 seconds

## Local preview
Open `index.html` directly in any browser. No build step needed.
